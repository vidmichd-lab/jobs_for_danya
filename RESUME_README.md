# Resume - ATS-Optimized CV

Это резюме оптимизировано для ATS (Applicant Tracking Systems) и соответствует стандартам европейских компаний.

## Особенности

✅ **ATS-оптимизация:**
- Чистая HTML-структура с семантическими тегами
- Стандартные названия секций
- Простой, читаемый формат без сложной графики
- Ключевые слова из индустрии дизайна

✅ **Формат:**
- Одна страница A4 (210mm × 297mm)
- Профессиональная типографика
- Четкая структура с измеримыми достижениями

✅ **Стандарты:**
- Хронологический порядок (от нового к старому)
- Четкие даты в формате MM/YYYY
- Конкретные результаты и метрики

### 2. Экспорт в PDF

**Вариант 1: Через браузер (рекомендуется)**
1. Откройте `resume.html` в браузере (Chrome, Firefox, Safari)
2. Нажмите `Ctrl+P` (Windows/Linux) или `Cmd+P` (Mac)
3. Выберите "Сохранить как PDF"
4. В настройках печати:
   - Размер: A4
   - Поля: Минимальные или Нет
   - Фоновая графика: Включить
5. Сохраните файл

**Вариант 2: Через командную строку (если установлен Puppeteer/Playwright)**
```bash
# Пример с Puppeteer (нужна установка)
node export-pdf.js
```

## Кастомизация

Если нужно изменить стиль:
1. Отредактируйте `resume.css`
2. Основные параметры:
   - `font-size` - размер шрифта
   - `margin-bottom` - отступы между секциями
   - `padding` - внутренние отступы

**Важно:** Убедитесь, что резюме помещается на одну страницу A4. Если контент не помещается, уменьшите:
- Размер шрифта (но не менее 9pt)
- Отступы между элементами
- Количество пунктов в описании опыта

## Поддержка

Если резюме не помещается на одну страницу:
1. Уменьшите `font-size` в `.job-responsibilities li` до 9pt
2. Уменьшите `margin-bottom` в `.experience-item` до 10pt
3. Сократите некоторые пункты в описании опыта
